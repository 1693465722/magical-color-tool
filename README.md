# Advantage

* 简洁、清新界面
* 取色后自动复制颜色值，支持rgba hex
* 支持调色
* 自定义快捷键（默认alt+q）随叫随到无需等待
* 随机切换颜色，放松心情
* 全屏后有彩蛋哦，等你来看
  
# Run
```
 yarn || npm i
 npm run dev
```
